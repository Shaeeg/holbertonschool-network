# Basics #0: OSI model

Introductory project on how network communication is described using the **OSI (Open Systems Interconnection)** model: layered design from physical transmission up to application protocols, as a **conceptual** framework (not something that “runs” the network by itself).

This directory contains the quiz answer file for the OSI model questions.

## Context

- **Lower layers** (e.g. layer 1): physical transmission (electrical, optical, radio).
- **Higher layers** (e.g. layer 7): application-level protocols (e.g. HTTP, SNMP).
- Later network projects focus especially on the **transport** layer (TCP/UDP) and the **network** layer (IP, ICMP).
